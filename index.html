Meus Projetos

Doce&Sabor - Confeitaria

Descrição do Projeto

O site Doce&Sabor foi desenvolvido com o objetivo de criar uma plataforma online moderna e atraente para uma confeitaria especializada na criação de bolos e doces personalizados. O design foi pensado para proporcionar uma experiência visual agradável e intuitiva, utilizando tecnologias e ferramentas modernas para garantir desempenho e responsividade.
